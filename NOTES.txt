Flask-Babel (localization)
--------------------------
* from flask.ext.babel import gettext
* prekladane texty obalit gettext() alebo _() v html
* extrakcia textov pre preklad: pybabel extract -F babel.cfg -o messages.pot app
* generovanie katalogu pre konkretny jazyk: pybabel init -i messages.pot -d app/translations -l es
* kompilacia prelozenych textov: pybabel compile -d app/translations
* $ python tr_init.py [sk|cs|es]
* $ python tr_update.py
* $ python tr_compile.py
